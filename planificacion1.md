
### Etapa 14. IMG a PDF

**Página:** `static/img-to-pdf.html`

**Descripción:** Convierte cada imagen subida a un solo archivo PDF.

**Interfaz de usuario:**
1. Zona de carga de los archivos
2. Configuración de calidad:
   - DPI: [72 | 150 | 300 | 600 | original] (slider o selector)
   - Mostrar tamaño estimado del resultado según DPI seleccionado
3. Disposicion en páginas:
   - ( ) Todas las imagenes posibles en cada página.
   - ( ) Una imagen por pagina
4. Vista previa de todas las páginas para poder ordenar con drag and drop.
5. Información: "X páginas → aproximadamente Y MB"
6. Botón "Convertir a PDF"
